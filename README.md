# asr-tengri-api

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/heyoSSam/asr-tengri-api
   cd asr-tengri-api
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the project:**
   ```bash
   uvicorn main:app --reload
   ```
